=======================================
  KIPiA СИСТЕМА УПРАВЛЕНИЯ ПРИБОРАМИ
=======================================

ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ИСПОЛЬЗОВАНИЮ

1. УСТАНОВКА
   - Запустите файл KIPiA_Management_***.exe
   - Следуйте инструкциям мастера установки
   - Рекомендуется оставить путь установки по умолчанию
   - Отметьте "Создать значок на рабочем столе" для удобства

2. ЗАПУСК ПРОГРАММЫ
   - Используйте ярлык на рабочем столе "KIPiA Management"
   - Или через меню Пуск -> KIPiA Management

3. СИНХРОНИЗАЦИЯ ДАННЫХ МЕЖДУ КОМПЬЮТЕРАМИ
   Для переноса данных между компьютерами:
   - На компьютере-источнике: скопируйте файл базы данных
     C:\Users\*****\AppData\Roaming\KIPiA_Management\data\kipia_management.db
   - На целевом компьютере: замените файл базы данных
     C:\Users\*****\AppData\Roaming\KIPiA_Management\data\kipia_management.db
   - Перезапустите программу

4. РАБОТА С ЛОГАМИ ПРИЛОЖЕНИЯ

   4.1. ГДЕ НАХОДЯТСЯ ЛОГИ
   - Файл логов: C:\Users\*****\AppData\Roaming\KIPiA_Management\logs\kipia-management.log
   - Папка с логами создается автоматически при первом запуске

   4.2. КАК ПРОСМАТРИВАТЬ ЛОГИ
   - Используйте файл ViewLogs.bat в папке установки программы.
   - Посмотреть логи можно из папки в меню Пуск.
   - Откройте файл логов в любом текстовом редакторе.

   4.3. ДЛЯ ЧЕГО НУЖНЫ ЛОГИ
   - Диагностика ошибок при запуске
   - Отслеживание работы приложения
   - Поиск причин сбоев
   - Техническая поддержка

5. ДОПОЛНИТЕЛЬНЫЕ СКРИПТЫ (в папке установки)

   5.1. DebugLauncher.bat
   - Запуск приложения с подробными отладочными логами
   - Используйте при проблемах с запуском
   - Показывает техническую информацию в консоли

   5.2. RunWithConsole.bat
   - Запуск приложения с отображением консоли
   - Видны все сообщения программы в реальном времени
   - Удобно для отслеживания работы

   5.3. ViewLogs.bat
   - Быстрый просмотр файла логов в Блокноте
   - Автоматически находит файл логов

   5.4. CheckEnvironment.bat
   - Проверка окружения системы
   - Показывает версию Java, пути установки
   - Проверяет наличие необходимых файлов

6. РЕШЕНИЕ ПРОБЛЕМ

   6.1. ПРОГРАММА НЕ ЗАПУСКАЕТСЯ
   - Запустите DebugLauncher.bat для диагностики
   - Проверьте установку Java с помощью CheckEnvironment.bat
   - Убедитесь, что в логах нет ошибок

   6.2. ГРАФИЧЕСКИЙ ИНТЕРФЕЙС НЕ ОТОБРАЖАЕТСЯ
   - Используйте RunWithConsole.bat для запуска
   - Проверьте сообщения об ошибках в консоли

   6.3. ПОТЕРЯ ДАННЫХ
   - Регулярно делайте резервные копии файла базы данных
   - Файл базы: C:\Users\*****\AppData\Roaming\KIPiA_Management\data\kipia_management.db

7. СИСТЕМНЫЕ ТРЕБОВАНИЯ
   - Java 23 или выше (ОБЯЗАТЕЛЬНО)
   - Windows 10/11 (64-битная)
   - 500 МБ свободного места на диске
   - 4 ГБ оперативной памяти

8. ЕСЛИ ПРОГРАММА НЕ ЗАПУСКАЕТСЯ
   - Убедитесь, что установлена Java 23 или выше
   - Скачайте Java с официального сайта:
     https://www.oracle.com/java/technologies/downloads/
   - Перезагрузите компьютер после установки Java

9. ТЕХНИЧЕСКАЯ ПОДДЕРЖКА
   При обращении в техническую поддержку предоставьте:
   - Файл логов (kipia-management.log)
   - Результат запуска CheckEnvironment.bat
   - Описание проблемы и шаги для воспроизведения

10. УДАЛЕНИЕ ПРОГРАММЫ
    - Через Панель управления -> Программы и компоненты
    - Или через меню Пуск -> Параметры -> Приложения
    - При удалении база данных и логи сохраняются в AppData

ПРИМЕЧАНИЕ: Все пользовательские данные (база данных, логи, настройки) 
хранятся в папке AppData и не удаляются при обновлении программы.
