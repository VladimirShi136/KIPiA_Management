Версия 1.0.0 (первая релиз‑версия)
Дата выпуска: 17.11.2025
Статус: первая рабочая сборка

Основные нововведения:

--Реализовано создание базы данных (БД) — базовая функциональность для хранения и управления данными приложения.

--Одна таблица приборов - два вида просмотра: общий список и отсортированный по местоположению. 

--Добавлена функция экспорта и импорта табличных данных в формате Microsoft Excel (.xlsx).

--Введён редактор схем — инструмент для визуального проектирования и редактирования схем приборов/систем.

--Реализован модуль формирования отчётов о приборах — генерация структурированных отчётов на основе данных из БД.

Значимость версии:
Первая стабильная сборка, содержащая базовый функционал для работы с данными, схемами и отчётностью.
